# Git Final Project

## Description
This repository contains the final project deliverables for the IBM Introduction to Git and GitHub course. It demonstrates essential version control concepts including repository creation, license selection, code of conduct enforcement, open-source contribution guidelines, branch management, merging, reverting, and bash scripting.

## Contents
- `README.md` - Project description and overview
- `LICENSE` - Apache 2.0 open-source license
- `CODE_OF_CONDUCT.md` - Community standards and pledge
- `CONTRIBUTING.md` - Guidelines for contributing to this project
- `simple-interest.sh` - Bash script to calculate simple interest
