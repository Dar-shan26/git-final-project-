# Simple Interest Calculator

A Bash script application to calculate simple interest based on user inputs: principal amount, annual rate of interest, and time period in years.

## Description
This repository is created for the IBM Introduction to Git and GitHub course final project. It contains a simple interest calculator bash script (`simple-interest.sh`) along with standard open-source documentation.

## Features
- Computes simple interest using the formula: Simple Interest = (Principal * Rate * Time) / 100
- Prompts user for input: principal amount, annual rate of interest, and time period in years.
- Outputs the calculated simple interest.

## Usage
Run the script using bash:
```bash
./simple-interest.sh
```
