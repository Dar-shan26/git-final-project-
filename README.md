# Simple Interest Calculator

A Bash script application to calculate simple interest based on user inputs: principal amount, annual rate of interest, and time period in years.

## Description
This repository contains the final project deliverables for the IBM Introduction to Git and GitHub course. It provides a simple interest calculator script (`simple-interest.sh`) along with open-source repository documentation including LICENSE, CODE_OF_CONDUCT.md, and CONTRIBUTING.md.

## Usage
To run the simple interest calculator script:
```bash
bash simple-interest.sh
```

### Input Fields:
- **Principal amount** (p)
- **Rate of interest per year** (r)
- **Time period in years** (t)

### Output:
- **Simple Interest** calculated as: `(p * r * t) / 100`
